# **App Name**: E-Commers V

## Core Features:

- Guest Product View: Allows guests to view products on the landing page but redirects to login/register upon product selection.
- User Product Management: Enables users to view product details, add items to the cart, view the cart, and proceed to checkout.
- User Order Tracking: Provides users with order history and profile access.
- Admin Product Control: Allows admins to manage products by adding, editing, and deleting items.
- Order Status Management: Enables admins to view incoming orders and update their status.
- Admin Dashboard: Presents admins with a dashboard for monitoring product performance and key metrics.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) for a sense of trust and sophistication.
- Background color: Very light lavender (#F0F8FF) for a clean, uncluttered feel.
- Accent color: Sunset Orange (#FF4500) to highlight important interactive elements.
- Body font: 'PT Sans', a sans-serif, for comfortable reading.
- Headline font: 'Playfair', a serif, for elegant titles (use 'PT Sans' for longer body text).
- Use simple, outlined icons for product categories and user actions.
- Subtle transitions and loading animations to enhance user experience.
